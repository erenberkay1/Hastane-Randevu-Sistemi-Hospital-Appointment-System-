# 🏥 National Health System / Milli Sağlık Sistemi

[TR] Dolu saatleri dinamik olarak filtreleyebilen gelişmiş bir randevu yönetim sistemidir.
[EN] An advanced appointment management system that dynamically filters booked slots.

---

## 🌟 Key Features / Öne Çıkan Özellikler

### [EN] English
- **Dynamic Doctor List:** Fetches real-time doctor data from the database.
- **Smart Slot Filtering:** Automatically hides booked hours for a specific doctor and date.
- **Appointment Lifecycle:** Create and cancel (Delete) appointments seamlessly.
- **Relational Queries:** Uses SQL JOINs to combine patient and doctor data.

### [TR] Türkçe
- **Dinamik Doktor Listesi:** Veritabanından gerçek zamanlı doktor verilerini çeker.
- **Akıllı Saat Filtreleme:** Belirli bir doktor ve tarih için dolu saatleri otomatik gizler.
- **Randevu Döngüsü:** Randevu oluşturma ve sorunsuz iptal etme (Silme).
- **İlişkisel Sorgular:** Hasta ve doktor verilerini birleştirmek için SQL JOIN kullanır.

---

## 🛠️ Tech Stack / Teknolojiler
- **Backend:** C# .NET Core API
- **Database:** SQLite
- **Frontend:** Dynamic DOM management, Async/Await Fetch API
